# MiddleOut Public Smoke

This repository is intentionally small. The default `main` branch documents the public smoke target; GitHub Pages serves the generated files from the `gh-pages` branch.

Open the live smoke page:
https://martin123132.github.io/MiddleOut-public-smoke/

Machine-readable latest pointer:
https://martin123132.github.io/MiddleOut-public-smoke/latest-public-smoke.json

The `gh-pages` branch contains only generated fixture HLS smoke artifacts used to prove that MiddleOut release evidence can be published and probed over public HTTP. Buyer media, source videos, credentials, and private MiddleOut repository contents are not published here.
